Introduction
the weighted rich-club effect tests whether a select group
of nodes share their strongest ties with each other in a
weighted network (Opsahl et al., 2008)

 80-20 rule  (Pareto, 1897)

high degree nodes form many ties with each other
(Borgatti and Everett, 1999; Colizza et al., 2006; Newman, 2002)


4+2+2+3+1+2 = 14
3+4+4+4+3+3 = 21
There are four initial steps:
1. Define certain nodes as prominent;
2. Calculate the total sum of weights attached
to ties among the prominent nodes;
3. Calculate the total sum of same number of ties,
 but the strongest ones in the network;
4. Find the ratio of point 2 and point 3. 
Rich-club effect
r  ---richness

For each value of r, we select the group (the club) of all
nodes whose richness is larger than r. We thus obtain a
series of increasingly selective clubs.

For each of these clubs, we count the number of links (E)  
connecting the members, and measure the sum of the
weights attached to these links
             
In the whole network,  we can find the the top E ranked weights
 on the links of the network
